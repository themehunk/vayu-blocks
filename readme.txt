== Description ==
The Vayu Blocks is an add-on plugin For Gutenberg Block Editor. Quickstart the Gutenberg editor with Powerful and elegant blocks to design stunning websites.
Free Vayu Blocks plugin that amplifies the default WordPress Gutenberg Editor with powerful blocks. 

### Blocks Available in Vayu

#### Advance Container [FREE]
Container block allows you to create visually consistent and organized sections within your content area.

#### Advance Heading [FREE]
Heading block is a fundamental content block used for creating and styling headings or titles within your posts or pages.

#### Advance Spacer [FREE]
Spacer block is used to create empty spaces between content blocks, improving visual separation and layout control.

#### Advance Button [FREE]
Easily design attractive buttons with Vayu Blocks advanced customizations.

#### Advance Product [FREE]
This enables you to seamlessly integrate your WooCommerce products into your content, in posts or pages.

== Installation ==

#### Automatic Installation

To download and install the Vayu blocks plugin, make sure you follow the steps below:

- Navigate to the WordPress  Dashboard > Plugins.
- Click the Add New button and type in the Search Bar "Vayu Blocks".
- When the plugin appears, click on the Install button and then on  Activate.

#### Manual Installation

- Log into your website admin page, then go to Appearance > Plugins. You will see all the plugins currently installed in your dashboard listed there.
- To install a new WordPress plugin, simply click on the Add New tab.
- Click the Upload Plugin button in the WordPress Dashboard.
- Click the Choose File button, select the ZIP archive of the Vayu Blocks plugin from your local computer and press the Install Now button.
- Click on the Activate button under the "Plugin installed successfully" message.
